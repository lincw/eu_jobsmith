## Release Decision

PASS for initial public release, with one documented model caveat: Codex CLI resume health may use conservative fallback when its raw output is not valid structured JSON. The endpoint now uses one structured attempt, pins app-driven Codex calls to low reasoning effort, and falls back on timeout/runtime failures, so the user-facing path completes with profile, progress, assessment, and done events instead of waiting through repeated long JSON-repair retries or surfacing a raw CLI timeout.
